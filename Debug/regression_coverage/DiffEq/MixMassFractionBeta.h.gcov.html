       2                 :            : /*!
       3                 :            :   \file      src/DiffEq/MixMassFractionBeta.h
       4                 :            :   \copyright 2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.
       5                 :            :   \brief     System of mix mass-fraction beta SDEs
       6                 :            :   \details   This file implements the time integration of a system of stochastic
       7                 :            :     differential equations (SDEs) with linear drift and quadratic diagonal
       8                 :            :     diffusion, whose invariant is the joint [beta
       9                 :            :     distribution](http://en.wikipedia.org/wiki/Beta_distribution). There are two
      10                 :            :     differences compared to the plain beta SDE (see DiffEq/Beta.h):
      11                 :            : 
      12                 :            :     - First, the parameters, b, and kappa are specified via functions that
      13                 :            :     constrain the beta SDE to be consistent with the turbulent mixing process.
      14                 :            :     In particular, the SDE is made consistent with the no-mix and fully mixed
      15                 :            :     limits. See, e.g., MixMassFractionBetaCoeffConst::update().
      16                 :            : 
      17                 :            :     - Second, there two additional random variables computed, the same as also
      18                 :            :     computed by the mass-fraction beta equation, see also
      19                 :            :     DiffEq/MassFractionBeta.h.
      20                 :            : 
      21                 :            :     In a nutshell, the equation integrated governs a set of scalars,
      22                 :            :     \f$0\!\le\!Y_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
      23                 :            :     \f[
      24                 :            :        \mathrm{d}Y_\alpha(t) = \frac{b_\alpha}{2}\left(S_\alpha - Y_\alpha\right)
      25                 :            :        \mathrm{d}t + \sqrt{\kappa_\alpha Y_\alpha(1-Y_\alpha)}
      26                 :            :        \mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N
      27                 :            :     \f]
      28                 :            :     with parameter vectors \f$b_\alpha = \Theta b'_\alpha > 0\f$, \f$
      29                 :            :     \newcommand{\irv}[1]{\langle{#1^2}\rangle} \kappa_\alpha = \kappa' \irv{x} >
      30                 :            :     0\f$, and \f$0 < S_\alpha < 1\f$. This is similar to DiffEq/Beta.h, but the
      31                 :            :     parameters, \f$b\f$ and \f$\kappa\f$ constrained. Here \f$
      32                 :            :     \newcommand{\irv}[1]{\langle{#1^2}\rangle}
      33                 :            :     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \Theta = 1 - \irv{x} /
      34                 :            :     [ \irmean{Y} (1-\irmean{Y}) ]\f$. The fluctuation about the mean, \f$
      35                 :            :     \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \irmean{Y} \f$, is defined as
      36                 :            :     usual: \f$ \newcommand{\irmean}[1]{{\langle{#1}\rangle}} x = Y - \irmean{Y}
      37                 :            :     \f$, and \f$b'\f$ and \f$ \kappa'\f$ are user-specified constants. Also,
      38                 :            :     \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued
      39                 :            :     [Wiener process](http://en.wikipedia.org/wiki/Wiener_process) with
      40                 :            :     independent increments. The invariant distribution is the joint beta
      41                 :            :     distribution. This system of SDEs consists of N independent equations. For
      42                 :            :     more on the beta SDE, see https://doi.org/10.1080/14685248.2010.510843.
      43                 :            : 
      44                 :            :     In addition to integrating the above SDE, there are two additional functions
      45                 :            :     of \f$ Y_\alpha \f$ are computed as
      46                 :            :     \f[ \begin{align}
      47                 :            :       \rho(Y_\alpha) & = \frac{ \rho_{2\alpha} }{ 1 + r_\alpha Y_\alpha } \\
      48                 :            :       V(Y_\alpha) & = \frac{1}{ \rho(Y_\alpha) }
      49                 :            :     \end{align} \f]
      50                 :            :     These equations compute the instantaneous mixture density, \f$ \rho \f$, and
      51                 :            :     instantaneous specific volume, \f$ V_\alpha \f$, for equation \f$ \alpha \f$
      52                 :            :     in the system. These quantities are used in binary mixing of
      53                 :            :     variable-density turbulence between two fluids with constant densities, \f$
      54                 :            :     \rho_1, \f$ and \f$ \rho_2 \f$. The additional parameters, \f$ \rho_2 \f$
      55                 :            :     and \f$ r \f$ are user input parameters and kept constant during
      56                 :            :     integration. Since we compute the above variables, \f$\rho,\f$ and \f$V\f$,
      57                 :            :     and call them mixture density and specific volume, respectively, \f$Y\f$,
      58                 :            :     governed by the beta SDE is a mass fraction.
      59                 :            : 
      60                 :            :     _All of this is unpublished, but will be linked in here once published_.
      61                 :            : */
