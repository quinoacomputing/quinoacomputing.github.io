2 /*!
3  \file src/DiffEq/MixNumberFractionBeta.h
4  \copyright 2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.
5  \brief System of mix number-fraction beta SDEs
6  \details This file implements the time integration of a system of stochastic
7  differential equations (SDEs) with linear drift and quadratic diagonal
8  diffusion, whose invariant is the joint [beta
9  distribution](http://en.wikipedia.org/wiki/Beta_distribution). There are two
10  differences compared to the plain beta SDE (see DiffEq/Beta.h):
11 
12  - First, the parameters, b, and kappa are specified via functions that
13  constrain the beta SDE to be consistent with the turbulent mixing process.
14  In particular, the SDE is made consistent with the no-mix and fully mixed
15  limits. See, e.g., MixNumberFractionBetaCoeffConst::update().
16 
17  - Second, there two additional random variables computed, the same as also
18  computed by the number-fraction beta equation, see also
19  DiffEq/NumberFractionBeta.h.
20 
21  In a nutshell, the equation integrated governs a set of scalars,
22  \f$0\!\le\!X_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
23  \f[
24  \mathrm{d}X_\alpha(t) = \frac{b_\alpha}{2}\left(S_\alpha - X_\alpha\right)
25  \mathrm{d}t + \sqrt{\kappa_\alpha X_\alpha(1-X_\alpha)}
26  \mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N
27  \f]
28  with parameter vectors \f$b_\alpha = \Theta b'_\alpha > 0\f$, \f$
29  \newcommand{\irv}[1]{\langle{#1^2}\rangle} \kappa_\alpha = \kappa' \irv{x} >
30  0\f$, and \f$0 < S_\alpha < 1\f$. This is similar to DiffEq/Beta.h, but the
31  parameters, \f$b\f$ and \f$\kappa\f$ constrained. Here \f$
32  \newcommand{\irv}[1]{\langle{#1^2}\rangle}
33  \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \Theta = 1 - \irv{x} /
34  [ \irmean{X} (1-\irmean{X}) ]\f$. The fluctuation about the mean, \f$
35  \newcommand{\irmean}[1]{{\langle{#1}\rangle}} \irmean{X} \f$, is defined as
36  usual: \f$ \newcommand{\irmean}[1]{{\langle{#1}\rangle}} x = X - \irmean{X}
37  \f$, and \f$b'\f$ and \f$ \kappa'\f$ are user-specified constants. Also,
38  \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued
39  [Wiener process](http://en.wikipedia.org/wiki/Wiener_process) with
40  independent increments. The invariant distribution is the joint beta
41  distribution. This system of SDEs consists of N independent equations. For
42  more on the beta SDE, see https://doi.org/10.1080/14685248.2010.510843.
43 
44  Similar to the number-fraction beta SDE (DiffEq/NumberFractionBeta.h), in
45  addition to integrating the above SDE, there are two additional functions
46  of \f$ X_\alpha \f$ are computed as
47  \f[ \begin{align}
48  \rho(X_\alpha) & = \rho_{2\alpha} ( 1 - r'_\alpha X_\alpha ) \\
49  V(X_\alpha) & = \frac{1}{ \rho_{2\alpha} ( 1 - r'_\alpha X_\alpha ) }
50  \end{align} \f]
51  These equations compute the instantaneous mixture density, \f$ \rho \f$, and
52  instantaneous specific volume, \f$ V_\alpha \f$, for equation \f$ \alpha \f$
53  in the system. These quantities are used in binary mixing of
54  variable-density turbulence between two fluids with constant densities, \f$
55  \rho_1, \f$ and \f$ \rho_2 \f$. The additional parameters, \f$ \rho_2 \f$
56  and \f$ r' \f$ are user input parameters and kept constant during
57  integration. Since we compute the above variables, \f$\rho,\f$ and \f$V\f$,
58  and call them mixture density and specific volume, respectively, \f$X\f$,
59  governed by the beta SDE is a number (or mole) fraction.
60 
61  _All of this is unpublished, but will be linked in here once published_.
62 */

139  //! \brief Advance particles according to the system of mix number-fraction
140  //! beta SDEs
141  //! \param[in,out] particles Array of particle properties
142  //! \param[in] stream Thread (or more precisely stream) ID
143  //! \param[in] dt Time step size
144  //! \param[in] moments Map of statistical moments
145  void advance( tk::Particles& particles,
146  int stream,
147  tk::real dt,
148  tk::real,
149  const std::map< tk::ctr::Product, tk::real >& moments )
150  {
151  // Update SDE coefficients
152  coeff.update( m_depvar, m_ncomp, moments, m_bprime, m_kprime, m_b, m_k );
153  // Advance particles
154  const auto npar = particles.nunk();
155  for (auto p=decltype(npar){0}; p<npar; ++p) {
156  // Generate Gaussian random numbers with zero mean and unit variance
157  std::vector< tk::real > dW( m_ncomp );
158  m_rng.gaussian( stream, m_ncomp, dW.data() );
159  // Advance all m_ncomp scalars
160  for (ncomp_t i=0; i<m_ncomp; ++i) {
161  tk::real& X = particles( p, i, m_offset );
162  tk::real d = m_k[i] * X * (1.0 - X) * dt;
163  d = (d > 0.0 ? std::sqrt(d) : 0.0);
164  X += 0.5*m_b[i]*(m_S[i] - X)*dt + d*dW[i];
165  // Compute instantaneous values derived from updated X
166  particles( p, m_ncomp+i, m_offset ) = rho( X, i );
167  particles( p, m_ncomp*2+i, m_offset ) = vol( X, i );
168  }
169  }
170  }
