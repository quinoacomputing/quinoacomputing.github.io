2 /*!
3  \file src/DiffEq/NumberFractionBeta.h
4  \copyright 2012-2015, J. Bakosi, 2016-2018, Los Alamos National Security, LLC.
5  \brief System of number-fraction beta SDEs
6  \details This file implements the time integration of a system of stochastic
7  differential equations (SDEs) with linear drift and quadratic diagonal
8  diffusion, whose invariant is the joint [beta
9  distribution](http://en.wikipedia.org/wiki/Beta_distribution). The main
10  difference compared to the plain beta SDE (see DiffEq/Beta.h), is that in
11  the number-fraction beta SDE the dependent variable, there are two
12  additional stochastic variables computed from the beta variables.
13 
14  In a nutshell, the equation integrated governs a set of scalars,
15  \f$0\!\le\!X_\alpha\f$, \f$\alpha\!=\!1,\dots,N\f$, as
16  \f[
17  \mathrm{d}X_\alpha(t) = \frac{b_\alpha}{2}\left(S_\alpha - X_\alpha\right)
18  \mathrm{d}t + \sqrt{\kappa_\alpha X_\alpha(1-X_\alpha)}
19  \mathrm{d}W_\alpha(t), \qquad \alpha=1,\dots,N
20  \f]
21  with parameter vectors \f$b_\alpha > 0\f$, \f$\kappa_\alpha > 0\f$, and \f$0
22  < S_\alpha < 1\f$. This is the same as in DiffEq/Beta.h. Here
23  \f$\mathrm{d}W_\alpha(t)\f$ is an isotropic vector-valued [Wiener
24  process](http://en.wikipedia.org/wiki/Wiener_process) with independent
25  increments. The invariant distribution is the joint beta distribution. This
26  system of SDEs consists of N independent equations. For
27  more on the beta SDE, see https://doi.org/10.1080/14685248.2010.510843.
28 
29  In addition to integrating the above SDE, there are two additional functions
30  of \f$ X_\alpha \f$ are computed as
31  \f[ \begin{align}
32  \rho(X_\alpha) & = \rho_{2\alpha} ( 1 - r'_\alpha X_\alpha ) \\
33  V(X_\alpha) & = \frac{1}{ \rho(X\alpha) }
34  \end{align} \f]
35  These equations compute the instantaneous mixture density, \f$ \rho \f$, and
36  instantaneous specific volume, \f$ V_\alpha \f$, for equation \f$ \alpha \f$
37  in the system. These quantities are used in binary mixing of
38  variable-density turbulence between two fluids with constant densities, \f$
39  \rho_1, \f$ and \f$ \rho_2 \f$. The additional parameters, \f$ \rho_2 \f$
40  and \f$ r' \f$ are user input parameters and kept constant during
41  integration. Since we compute the above variables, \f$\rho,\f$ and \f$V\f$,
42  and call them mixture density and specific volume, respectively, \f$X\f$,
43  governed by the beta SDE is a number (or mole) fraction, hence the name
44  number-fraction beta.
45 
46  _All of this is unpublished, but will be linked in here once published_.
47 */

118  //! \brief Advance particles according to the system of number-fraction beta
119  //! SDEs
120  //! \param[in,out] particles Array of particle properties
121  //! \param[in] stream Thread (or more precisely stream) ID
122  //! \param[in] dt Time step size
123  void advance( tk::Particles& particles,
124  int stream,
125  tk::real dt,
126  tk::real,
127  const std::map< tk::ctr::Product, tk::real >& )
128  {
129  // Advance particles
130  const auto npar = particles.nunk();
131  for (auto p=decltype(npar){0}; p<npar; ++p) {
132  // Generate Gaussian random numbers with zero mean and unit variance
133  std::vector< tk::real > dW( m_ncomp );
134  m_rng.gaussian( stream, m_ncomp, dW.data() );
135  // Advance all m_ncomp scalars
136  for (ncomp_t i=0; i<m_ncomp; ++i) {
137  tk::real& X = particles( p, i, m_offset );
138  tk::real d = m_k[i] * X * (1.0 - X) * dt;
139  d = (d > 0.0 ? std::sqrt(d) : 0.0);
140  X += 0.5*m_b[i]*(m_S[i] - X)*dt + d*dW[i];
141  // Compute instantaneous values derived from updated X
142  particles( p, m_ncomp+i, m_offset ) = rho( X, i );
143  particles( p, m_ncomp*2+i, m_offset ) = vol( X, i );
144  }
145  }
146  }
